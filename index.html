Spaced-Repetition

Language learning app

Designed and developed a fully responsive full stack react app, with authentication. This web application uses spaced repetition to help people memorize the Italian language. The app shows the word in Italy and asks you to remember the English translation of the word. You must log in to the program to start learning as a registered user. The words you forget more often are more often reflected. Once each word is mastered, each word is asked increasingly less frequently.

Technologies - React, Vanilla CSS, Node package manager, Node.js, Express, PostgreSQL, Cypress - (testing).
